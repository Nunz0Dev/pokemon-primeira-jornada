# 🎮 Pokémon Primeira Jornada (Versão 1)

Meu primeiro mini jogo em C inspirado em Pokémon.

## 🧠 Funcionalidades
- Escolha de Pokémon inicial
- Cadastro de jogador
- Verificação de idade
- Encontro com Pokémon selvagem
- Sistema de captura com chance aleatória

## 🛠️ Tecnologias
- Linguagem C
- stdio.h
- stdlib.h
- time.h
- string.h

## 🚀 Como executar

Compile:

gcc jogo.c -o jogo

Execute:

./jogo
